# Weather App

A simple weather application built using HTML, CSS, JavaScript, Fetch API, and OpenWeather API.

## Features

- Search weather by city name
- Displays temperature, humidity, and weather condition
- Handles invalid city names
- Handles API/network errors
- Uses asynchronous JavaScript with Fetch API

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- OpenWeather API
- Bootstrap
